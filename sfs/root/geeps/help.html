GEEPS help information:


Introduction:

GEEPS provides access to each drive that it sees defined on your computer.  Because most Windows computers have more than one partition to the drive, you will see drives you did not know you had.  These typically have emergency / rescue files on them or are configured as a Windows swap file.  


Examining drive contents:

The icons you will see in GEEPS don’t look like Windows, because they aren’t.  However, folders are very easily recognized, as are most other file types.  If you are not sure, click once on a file, and GEEPS will open it for you.  GEEPS can open nearly any type of file, so you will be able to quickly see what this file contains and if you are interested in keeping it.


Moving files:

When you plug an external USB drive in, a different icon with a USB symbol on it will show up to the right of the PC drives.  This is where you will drag the files you want to save to.  Click on it to open it.  You can drag individual files, a folder of files, or folders of folders of files.  Better to copy more files if in doubt, especially if your plan is to wipe the drive when you are done.


Wiping drives:

Before using the wipe command, it is best to close the external drive window, remove the drive from the GEEPS computer and plug it into a pc or mac you want to move the data to. Confirm that you can copy files from it and that the files are as they should be.  

Now, with the external drive removed, go to the menu in the upper left of the GEEPS screen and select wipe drive.  A message will ask you to select a drive to continue.  Click on the drive that you want to wipe.  If you want to wipe all of them, just do them sequentially.  If your pc has partitions that don’t contain personal data and look like they might be system restore or swap drives, there is probably no need to wipe those drives.

GEEPS writes to each sector of the drive, truly erasing data as opposed to removing data indexes, which hides data but someone that knows what they are doing can still retrieve your data.  Because of this, wiping a drive can take some time.  Please allow plenty of time for the operation to continue.  GEEPs will return to the main screen after a wipe operation is completed, so as long as you see the wipe screen up, the system is still working.


Extras:

GEEPS has a couple of extra features that can come in handy.  One is a browser that allows you to access the internet if the pc running GEEPS is connected to the internet.  A wired network will be connected to automatically in most cases.  There is currently no way for you to enter wireless information in to allow GEEPS to connect to the internet wirelessly.


A console utility is also available for use by those that have some knowledge of Linux.


Other uses:

GEEPS is very capable, and can be utilized for a variety of purposes related to gaining access to information on hard drives that cannot be accessed otherwise.  Forgotten passwords, abandoned profiles and other types of issues can be easily resolved.
